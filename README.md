# webpack-reload-test

支援 SCSS、JS 熱載入，有開發模式與產出（build）模式兩種設定，詳見 webpack.config.js 與 webpack.build.config.js

## 使用方法

安裝必須模組：

```npm install
```

啟動開發模式：

```npm run dev
```
瀏覽器打開 (http://127.0.0.1:8080) 可以享受熱載入的功能


產出模式：

```npm run build
```
